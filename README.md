# Генератор описаний товаров для маркетплейсов с помощью ИИ

![Python](https://img.shields.io/badge/python-3.11-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![Status](https://img.shields.io/badge/status-active-brightgreen.svg)

Скрипт на Python, который автоматически создает продающие описания для товаров на основе названия и характеристик, используя API ChatGPT. Результат сохраняется в Excel-файл. Проект создан для портфолио и демонстрирует навыки работы с нейросетями, автоматизацией и обработкой данных.

## Особенности

- **Пакетная обработка** – можно обработать сразу все товары из Excel-файла.
- **Настраиваемый промпт** – легко изменить стиль описания под свои нужды (например, "креативно", "формально", "для Wildberries").
- **Обработка ошибок** – программа корректно реагирует на таймауты и проблемы с сетью, не падает при сбоях.
- **Поддержка разных API** – работает с CometAPI, APIYI и напрямую с OpenAI (через прокси-сервисы для пользователей из РФ).
- **Гибкость модели** – можно выбрать модель (GPT-3.5, GPT-4 и другие).

## Технологии

- **Python 3.11**
- **pandas**, **openpyxl** – работа с Excel-файлами
- **requests** – отправка запросов к API
- **OpenAI API / CometAPI / APIYI** – доступ к языковым моделям

## Установка и запуск

### 1. Клонируйте репозиторий

bash
git clone https://github.com/lirikaqq/project-with-description-AI.git
cd project-with-description-AI

## 2. Установите зависимости
Рекомендуется использовать виртуальное окружение (опционально, но желательно):

bash
python -m venv venv
source venv/bin/activate  # для Linux/Mac
venv\Scripts\activate     # для Windows 

Затем установите пакеты:

bash
pip install -r requirements.txt

## 3. Получите API-ключ
Для работы скрипта нужен доступ к API нейросети. Если вы в России, используйте прокси-сервисы:
CometAPI (рекомендую)
APIYI
Зарегистрируйтесь, пополните баланс (достаточно 100–200 рублей) и получите API-ключ.

## 4. Настройте скрипт
Откройте файл generate_descriptions.py и вставьте ваш ключ в переменную API_KEY. Также можно изменить BASE_URL (в зависимости от выбранного сервиса) и другие параметры:


API_KEY = "ваш_ключ"

BASE_URL = "https://api.cometapi.com/v1"  # или "https://api.apiyi.com/v1"

MODEL = "gpt-3.5-turbo"  # можно gpt-4, если доступно


Важно: Никогда не выкладывайте свой ключ в открытый доступ! Убедитесь, что файл .gitignore содержит *.key и .env, если вы храните ключи отдельно.

## 5. Подготовьте входной файл
Создайте в папке проекта файл goods.xlsx с двумя колонками:

name – название товара

features – характеристики (например, цвет, размер, материал)

Пример заполнения можно посмотреть в файлах !example_input.xlsx и !example_output.xlsx соответственно.

## 6. Запустите скрипт
bash
python generate_descriptions.py
После выполнения в той же папке появится файл goods_with_descriptions.xlsx с добавленной колонкой description, содержащей сгенерированные описания.

Пример
Входные данные (goods.xlsx):

name	features
Кроссовки Nike Air Max	черные, размер 42, подошва из пены
Смартфон Samsung A54	128GB, зеленый, экран 120Hz

Выходные данные (goods_with_descriptions.xlsx):
description
Откройте для себя комфорт и стиль с кроссовками Nike Air Max...
Мощный смартфон Samsung A54 с большим экраном и отличной камерой...
